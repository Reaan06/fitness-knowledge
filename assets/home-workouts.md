# Home Workouts — Complete Guide

## 1. Bodyweight Training

### Push Exercises
- **Wall Push-ups**: Beginner, 3x15
- **Knee Push-ups**: Intermediate, 3x12
- **Standard Push-ups**: Advanced, 4x10
- **Decline Push-ups**: Feet elevated, 3x10
- **Diamond Push-ups**: Triceps emphasis, 3x8
- **Pike Push-ups**: Shoulders, 3x10
- **Archer Push-ups**: Unilateral strength, 3x8 each

### Pull Exercises
- **Doorframe Rows**: Using towel or doorframe
- **Inverted Rows**: Under sturdy table
- **Chin-ups**: Doorway bar or pull-up bar
- **Towel Rows**: Loop towel around pole
- **Backpack Rows**: Weighted pulls

### Squat Variations
- **Air Squat**: 4x15
- **Pause Squat**: 3x12 (3-second hold)
- **Jump Squats**: 3x10 (power)
- **Pistol Squats**: 3x5 each (advanced)
- **Bulgarian Split Squat**: Rear foot on chair
- **Sumo Squat**: Wide stance, 3x15

### Lunge Variations
- **Forward Lunge**: 3x10 each
- **Reverse Lunge**: 3x10 each
- **Walking Lunge**: Space permitting
- **Lateral Lunge**: 3x10 each
- **Curtsy Lunge**: 3x10 each
- **Lunge Jumps**: 3x8 each (power)

### Core Exercises
- **Plank**: 3x45 seconds
- **Side Plank**: 3x30 seconds each
- **Dead Bug**: 3x10 each
- **Bird Dog**: 3x10 each
- **Bicycle Crunches**: 3x15 each
- **Leg Raises**: 3x12
- **Mountain Climbers**: 3x20

## 2. Minimal Equipment Workouts

### Dumbbell Only
- **Upper Body**: Curls, press, rows, flyes
- **Lower Body**: Goblet squats, lunges, deadlifts
- **Full Body**: Complexes (string exercises together)

### Kettlebell Only
- **Swing**: 5x20 (power)
- **Clean & Press**: 4x8 each
- **Goblet Squat**: 4x12
- **Turkish Get-up**: 3x3 each (skill)

## 3. HIIT Workouts at Home

### Tabata Protocol (4 minutes)
- 20 seconds work / 10 seconds rest × 8 rounds
- Exercise examples: Squat jumps, burpees, mountain climbers

### 20-Minute HIIT
- 40 seconds work / 20 seconds rest
- 5 exercises × 4 rounds
- Rest 1 minute between rounds

### EMOM (Every Minute on the Minute)
- 10 minutes total
- Minute 1: 10 burpees
- Minute 2: 15 squats
- Minute 3: 10 push-ups
- Continue alternating

## 4. Workout Programs

### Beginner (Weeks 1-4)
**Frequency**: 3 days/week, full body

**Day 1**:
- Squats: 3x12
- Push-ups (knee): 3x10
- Plank: 3x30 seconds
- Glute bridges: 3x15
- Rest: 60 seconds

**Day 2**:
- Lunges: 3x10 each
- Inverted rows: 3x10
- Side plank: 3x20 seconds each
- Step-ups: 3x10 each
- Rest: 60 seconds

**Day 3**:
- Sumo squats: 3x12
- Pike push-ups: 3x8
- Dead bug: 3x10 each
- Calf raises: 3x15
- Rest: 60 seconds

### Intermediate (Weeks 5-8)
**Frequency**: 4 days/week, upper/lower split

**Upper 1**:
- Push-ups: 4x12
- Inverted rows: 4x10
- Pike push-ups: 3x10
- Bicep curls (band): 3x12
- Tricep dips: 3x12
- Rest: 60 seconds

**Lower 1**:
- Squats: 4x15
- Bulgarian split squats: 3x10 each
- Romanian deadlift (band): 3x12
- Calf raises: 3x15
- Plank: 3x45 seconds
- Rest: 60 seconds

**Upper 2**:
- Diamond push-ups: 4x10
- Chin-ups: 4x max
- Lateral raises (band): 3x12
- Hammer curls (band): 3x12
- Overhead tricep extension: 3x12
- Rest: 60 seconds

**Lower 2**:
- Jump squats: 4x10
- Walking lunges: 3x12 each
- Glute bridges: 3x15
- Single-leg deadlift: 3x10 each
- Side plank: 3x30 seconds each
- Rest: 60 seconds

### Advanced (Weeks 9-12)
**Frequency**: 5-6 days/week, push/pull/legs

**Push**:
- Decline push-ups: 4x12
- Pike push-ups: 4x10
- Diamond push-ups: 3x10
- Tricep dips: 3x12
- Lateral raises: 3x12
- Rest: 45 seconds

**Pull**:
- Chin-ups: 5x max
- Inverted rows: 4x12
- Band pull-aparts: 3x15
- Bicep curls: 3x12
- Superman: 3x15
- Rest: 45 seconds

**Legs**:
- Pistol squats: 4x5 each
- Bulgarian split squats: 4x10 each
- Jump squats: 4x10
- Glute bridges: 3x15
- Calf raises: 3x20
- Plank: 3x60 seconds
- Rest: 45 seconds

## 8. Tips for Home Training

### Environment
- Clear space for movement
- Good ventilation
- Motivating music
- Timer visible

### Consistency
- Same time daily
- Schedule like appointments
- Track workouts
- Celebrate progress

### Progression
- Increase reps first
- Then add sets
- Then add resistance
- Then increase difficulty

### Recovery
- Rest days between intense sessions
- Sleep 7-9 hours
- Hydrate properly
- Stretch regularly

## 9. Common Home Training Mistakes

### Mistake 1: No Warm-up
- **Problem**: Increased injury risk
- **Solution**: 5-10 minutes dynamic warm-up

### Mistake 2: Poor Form
- **Problem**: Injuries, no progress
- **Solution**: Film yourself, use mirrors, slow down

### Mistake 3: No Progression
- **Problem**: Plateaus, boredom
- **Solution**: Follow progressive programs

### Mistake 4: Skipping Rest Days
- **Problem**: Overtraining, burnout
- **Solution**: Schedule recovery

### Mistake 5: Unrealistic Expectations
- **Problem**: Frustration, quitting
- **Solution**: Set realistic goals, track small wins
