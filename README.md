# Fitness Knowledge Skill

A comprehensive, evidence-based fitness knowledge base for the Vita app. This skill provides detailed information on training, nutrition, recovery, and all aspects of fitness.

## 🎯 What is this?

This is an LLM-first skill designed to provide accurate, evidence-based fitness information. It covers everything from basic exercises to advanced periodization, ensuring users get safe, effective guidance.

## 📚 Contents

### Core Modules

| Module | Description |
|--------|-------------|
| **Training Types** | Complete guide to all training modalities (strength, cardio, functional, plyometrics, etc.) |
| **Nutrition & Macros** | Calories, macros, carbs, protein, fats, micronutrients, hydration |
| **Sports Nutrition** | Advanced nutrition timing, supplements, special diets, weight management |
| **Gym Techniques** | All gym exercises with proper form, cues, and common mistakes |
| **Functional Training** | Movement patterns, core stability, balance, power development |
| **Home Workouts** | Bodyweight training, minimal equipment, HIIT at home |
| **Vita Workouts** | App-specific workout types and programs |
| **Recovery & Rest** | Sleep optimization, active recovery, deload weeks |
| **Supplements** | Evidence-based supplement guide (what works, what doesn't) |
| **Periodization** | Programming principles, training phases, periodization types |
| **Common Mistakes** | Frequent errors and how to fix them |
| **Equipment Guide** | Complete gym equipment reference |
| **Injury Prevention** | Injury prevention strategies, rehabilitation principles |
| **Mental Fitness** | Mindset, motivation, goal setting, stress management |
| **Tracking Metrics** | What to track and how to measure progress |

## 🚀 Features

- **Evidence-based**: All information backed by scientific research
- **Safety-first**: Includes safety considerations and contraindications
- **Progressive**: Provides progression and regression options for exercises
- **Comprehensive**: Covers all aspects of fitness
- **Vita-integrated**: Designed specifically for the Vita app

## 🛠️ Usage

### For AI Agents

1. Load the skill when fitness-related questions are asked
2. Read the relevant asset file based on the user's question
3. Provide comprehensive, evidence-based information
4. Always include safety considerations
5. Offer modifications for different fitness levels

### For Vita App Integration

This skill is designed to integrate with the Vita fitness app:
- Provides knowledge base for workout recommendations
- Offers nutrition guidance
- Supports recovery protocols
- Enables progress tracking

## 🎓 Key Principles

1. **Safety First**: Never give medical advice, always recommend professional consultation
2. **Evidence-Based**: Use scientific research, not bro-science
3. **Progressive**: Always provide progression and regression options
4. **Individual**: Consider age, fitness level, goals, and limitations
5. **Consistent**: Emphasize consistency over intensity
